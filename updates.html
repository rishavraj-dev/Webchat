Webchat Update Log

See what's new, improved, and coming soon!

Version 3.0.0 (July 6, 2025) - Launched!

  • Public launch of Webchat.
  • A major improved UI.
  • Added the support for videos.
  • Added support for direct URL sharing.
  • Don't forget to follow the official channel Web Chat Official.
  • Streamlined User Interface: Experience a majorly improved and intuitive UI for seamless navigation.
  • Enhanced Media Sharing: Share images, PDFs, and now videos directly within your chats.
  • Direct URL Support: Easily share and preview web links directly in messages.
  • Advanced Group & Channel Management: Comprehensive tools for creating, managing, and moderating private and public groups and channels.
  • Message Control: Enjoy features like easy replies, one-click forwarding, and improved context menus for better chat management.
  • Personalization & Device Management: Customize your profile with pictures and manage your logged-in devices from a central dashboard.
  • Robust Security: All your files and conversations are securely protected from unauthorized access.
  • Optimized Performance: Smoother navigation, faster message loading, and a more robust search experience.
  • Official Channel Integration: Stay connected and informed by following the official "Web Chat Official" channel for announcements and support.

Version 2.5.0 (July 2, 2025) - Major Feature Rollup!

  • Enhanced Media & File Management: Improved UI for preview of selected files, added support for major image formats, and seamless file selection and sharing.
  • Advanced Messaging Features: Easy replies and one-click message forwarding with preview; improved context menus for messages (e.g., "Reply," "Message Info"); and the forward icon is now always visible and uses a reliable SVG.
  • Comprehensive Group Capabilities: Introduction of private groups for exclusive conversations, along with expanded and full group management features for both private and public groups.
  • User Experience & Dashboard Improvements: UI improvements for Hamburger menu and a dedicated 'Chats' button; more robust search functionality; dashboard updates including option to log out of other devices; edit options moved to settings; and the ability for users to set their profile pictures for customization.
  • Stability & Fixes: Previous rollback issues (from Version 2.0.0) successfully fixed and re-released.

Version 2.0.0 (June 29, 2025) - Major Release!

  • Expanded Media Support: Images, PDF, and rich text.
  • Messages auto-delete after 24 hours for privacy.
  • Enhanced File Security: Files cannot be accessed by unauthorized users through direct links or any other means.
  • File Preview: File preview is now available before sending, ensuring you share the right content.
  • Improved Group Chats: Groups now display sender names, and you can easily start a personal chat with a sender by clicking on their name.
  • Dashboard (View-Only): A new dashboard is now available, providing a view-only overview of your activity.

Version 1.1.0 (June 28, 2025)

  • Public groups: Create and join public communities!
  • Improved chat message loading for smoother experience.

Version 1.0.0 (June 28, 2025)

  • Initial Launch!
  • Personal chats for one-on-one secure conversations.
  • Privacy-focused architecture implemented.
  • Messages auto-delete after 24 hours for privacy.
