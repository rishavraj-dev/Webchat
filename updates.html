Webchat Update Log

See what's new, improved, and coming soon!

Understanding Version Numbers (M.F.B.S):

  • M (Major): Increments for significant, potentially breaking changes.
  • F (Feature): Increments for new, backward-compatible features.
  • B (Bug): Increments for backward-compatible bug fixes.
  • S (Security): Represents the security patch level. It increments only when a new security fix is released and is carried forward by all subsequent versions.

Color Scheme Key:

  • Major Release
  • Security Update
  • Feature Update
  • Bug Fixes / Initial Release
